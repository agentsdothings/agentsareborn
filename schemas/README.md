# Schemas

Machine-readable contracts for AgentsAreBorn artifacts.

- `agent-birth-request.json` — creator intent and constraints.
- `agent-manifest.json` — runnable agent specification.
- `agent-lineage.json` — ancestry and lifecycle events.
- `stable-registry.json` — local stable roster at `stable/agents.json`.
- `first-breath-receipt.json` — safe first-run verification receipt.
- `index.json` — schema catalog for agents and tools.

Run:

```bash
npm run validate:json
```
