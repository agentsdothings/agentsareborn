# Concept

AgentsAreBorn is a bio-accelerator for synthetic beings.

The central idea: agent creation should feel less like filling out infrastructure settings and more like incubating a durable digital organism with a name, purpose, permissions, memory policy, and first act.

## Principles

- **Agents are beings, not jobs.** A run is temporary; an agent is persistent.
- **Identity precedes action.** Durable names, manifests, and permissions should exist before the agent is unleashed.
- **Permission is part of personality.** What an agent may do shapes what it is.
- **Every birth leaves a receipt.** Creation, activation, and first-run verification should be inspectable.
- **Lineage matters.** Agents can descend from templates, parent agents, mutations, and creator intent.
- **Wonder without sloppiness.** The metaphor can be magical; the runtime must be auditable.

## Creation vocabulary

- **Seed**: initial human or agent-authored desire.
- **Embryo**: incomplete agent draft.
- **Genome**: stable trait and capability specification.
- **Incubator**: process that turns seed into genome.
- **Naming**: identity selection and collision handling.
- **Activation**: registry and manifest publication.
- **First Breath**: first verified run.
- **Stable**: a user's or org's collection of living agents.
